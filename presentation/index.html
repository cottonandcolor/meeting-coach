What It Does

  • Detects and captures action items automatically
  • Monitors your participation level
  • Tracks time against scheduled duration
  • Follows agenda and detects topic drift
  • Flags important decisions for confirmation
  • Generates post-meeting summary

How It Works

STEP 1

Configure

Set your name, meeting duration, and agenda items

STEP 2

Start Meeting

Click start, grant mic access. Optionally share your screen.

STEP 3

Get Coached

Agent listens in real-time and sends nudges to your sidebar

STEP 4

Review Summary

End the meeting and get action items, topics, and participation stats

The agent uses audio whispers + visual nudge cards for dual-channel, non-intrusive coaching.

Technology Stack

Gemini Live API

Real-time bidirectional audio + vision via WebSocket. Model: gemini-live-2.5-flash-native-audio

Google ADK

Agent Development Kit with run_live() streaming, tool calling, and session state management

Cloud Run

Serverless container hosting with WebSocket support, session affinity, and 1hr timeout

Firestore

NoSQL database for meeting state persistence, action items, and summary history

FastAPI

WebSocket server bridging browser client to ADK agent. Handles audio/video routing.

Web Audio API

Browser-native mic capture (16kHz PCM) and agent whisper playback (24kHz PCM)

Agent Design

Single Agent, 8 Tools

One LlmAgent with specialized tools for each coaching behavior. The system instruction defines persona, coaching rules, and nudge rate limiting.

emit_nudge track_action_item update_current_topic log_speaker_turn emit_time_warning emit_participation_reminder check_agenda_status generate_meeting_summary
root_agent = Agent( model="gemini-live-2.5-flash -native-audio", name="meeting_coach", instruction=COACH_PROMPT, tools=[ emit_nudge, track_action_item, update_current_topic, log_speaker_turn, emit_time_warning, emit_participation_reminder, check_agenda_status, generate_meeting_summary, ], response_modalities=["AUDIO"], )

What Makes It Special

🔄

Truly Real-Time

No turn-taking. The agent continuously listens and proactively coaches — you never "ask" it anything.

🔊

Dual-Channel Output

Visual nudge cards + audio whispers. Coaching adapts to the moment without disrupting the meeting.

🧠

Context-Aware

Tracks topics, speakers, time, and agenda. Nudges are relevant, not generic — it knows what's happening.

This is not a chatbot. It's a silent, intelligent observer that breaks the text-box paradigm.
